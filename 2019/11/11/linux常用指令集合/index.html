linux常用指令集合

简陋，并且随时增加

赋予权限

chmod 777 ./* -R

VIM

:w //save
:wq //save&quit
:q! //强退不保存
zz //有修改save&quit；无修改退出
:w filename //另存为
:r filename //将filename文件内容读到光标处

gg //文档第一行

远程传输

scp root@ip地址:/usr1/code/……/target空格./

压缩和解压缩

zip -r filename ./ //压缩到当前文件夹
zip -r filename

正则表达式搜索文件内容

grep -rn “xxxx” //r表示递归，n表示行号

tips

执行.sh文件时看一下，是否可以改成j8或者j24，使用多线程提高编译速度。
